# Contributing to AI Personal Trainer
Thank you for your interest in contributing! I encourage your help to make this project.

## How to Contribute
1. **Fork the repository.**
2. **Create a new branch** for your feature or bugfix.
3. **Write clear, concise commit messages.**
4. **Run tests** to ensure nothing is broken.
5. **Submit a pull request** describing your changes and why they are needed.

## Code Style
- Follow PEP8 for Python code.
- Use meaningful variable and function names.
- Write comments and docstrings where necessary.

## Reporting Issues
- Use GitHub Issues to report bugs or request features.
- Provide detailed descriptions and steps to reproduce.

## Code of Conduct
Please respect all community members and follow the [Code of Conduct](CODE_OF_CONDUCT.md).
